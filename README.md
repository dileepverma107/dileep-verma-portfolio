# Getting Started Locally

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/dileepverma107/dileep-verma-portfolio
   ```

2. Move to the cloned directory

   ```bash
   cd portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   ```
